# ta‘s backup file

这是 Gongxing Academy 当前版本的分类备份。

## 目录结构

- `html/index.html`：学习题库主页面
- `html/games.html`：游戏奖励中心页面
- `js/games.js`：六款小游戏逻辑
- `data/`：数学、英语、科学及拓展题库
- `assets/`：游戏图片资源
- `tools/embed-banks.mjs`：将 `data/` 题库重新嵌入主页面的工具

## 使用方式

直接打开 `html/index.html` 即可进入学习页面。题库已经内嵌，因此主页面可以离线打开。

游戏页面依赖 `js/` 和 `assets/`，请保留整个主文件夹的目录结构。通过学习页面用积分兑换游戏时间后进入游戏中心。
